Exhibition Statement

Intro to Computer Vision in SAIC's Art and Technology Studies department is a technically rigorous studio course that introduces students to computer vision â€” an interdisciplinary field of study around developing hardware and software that allows computers to see and "understand" the world around them. For as long as artists have incorporated computers into their practices, they have utilized computer vision technologies, whether through interactive installations that react to bodies captured through cameras, or more recently in experiments with 'Artificial Intelligence' using machine learning and deep neural networks.

This course teaches both "traditional" computer vision and contemporary machine-learning-based approaches. The semester began with learning how to use code to manipulate images and video pixel by pixel to make interactive experiences using webcams and color+depth sensors like the Microsoft Kinect. Then, the course pivoted to deep learning and Generative Adversarial Networks (GANs), where the technical focus shifted from code writing towards the meticuluous assembly of data sets.

While this course embraces the creative possibilities afforded by these tools, it also critically engages with the realities of working with technologies that surveil, control, and further entrench the harmful biases embedded in our culture. Neural networks "learn" to synthesize images by 'training' on a data set of often thousands of images. As artists working with these technologies, our critical entry point typically begins with the composition of our data sets. Where did this data come from? What does it mean for my work to use it? What sorts of biases exist in these data sets, and are those biases problematic?

In this collection of final projects, these artists have all experimented with the generative possibilities of deep neural networks. Some have used these tools to look outward, using images scraped from external sources to reimagine the world around them. Others looked inward, using these tools to create deeply personal works that examine how these technologies shape who we are. Although these works are presented as "final projects", many are still considered to be in progress.
